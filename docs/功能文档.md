### 3.2 游戏页（GamePage）—— 核心玩法

**文件：** `pages/GamePage.ets`  
**ViewModel：** `viewmodel/GameViewModel.ets`  
**定时器：** `viewmodel/TimerViewModel.ets`

#### 3.2.1 核心流程

```
加载题目 → 显示拼音首字母提示 → 用户输入答案 → 判判对错
   ↓                                     ↓
答错：errorCount++，hintLevel++        答对：score++，consecutiveCorrect++，播放动画
   ↓                                     ↓
提示逐步揭示                           进入 ResultPage 展示出处
   ↓
errorCount ≥ MAX_ERROR_COUNT → 显示正确答案，结束本题
```

#### 3.2.2 功能清单

| 功能 | 说明 |
|------|------|
| 拼音首字母提示 | 格式：`LFFW`（连续字母，逐步揭示） |
| 释义提示 | 显示成语释义/解释 |
| 逐步提示系统（F-08） | hintLevel=0 → "LF"；=1 → "LFF"；=2 → "LFFW"；≥3 → 完整答案 |
| 答案输入 | 文字输入框，支持模糊匹配（忽略标点） |
| 提交判判 | 答对/答错立即反馈，音效 + 动画 |
| 计时模式 | 60 秒倒计时，计时器组件 `TimerView` |
| 闯关模式 | 连续答对 5 题通关，答错一题重新开始当前关 |
| 错误次数限制 | 单题最多答错 3 次，超限显示正确答案 |
| 提示扣分 | 每次使用提示扣 2 分（SCORE_PENALTY_HINT） |
| 答对动画 | 烟花/星星动画（ResultPage），≤1 秒 |
| 成语出处展示 | 答对后在 ResultPage 显示来源（如《史记》） |

#### 3.2.3 提示系统详细逻辑（PRD F-08）

以 4 字成语「龙飞凤舞」为例：

| hintLevel | 显示内容 | 说明 |
|-----------|-----------|------|
| 0 | `LF` | 初始状态：显示前 2 个首字母 |
| 1 | `LFF` | 第 1 次提示：显示前 3 个首字母 |
| 2 | `LFFW` | 第 2 次提示：显示所有 4 个首字母 |
| ≥3 | `龙飞凤舞` | 第 3 次提示：显示完整答案 |

通用逻辑（N 字成语）：
- 初始显示前 2 个首字母
- 每次提示多揭示 1 个首字母
- hintLevel ≥ N-1 时显示完整答案
