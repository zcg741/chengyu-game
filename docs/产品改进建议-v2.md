# 妙语成语 - 产品改进建议（重新审视）

> 视角：真实用户角度评价  
> 日期：2026-05-11  
> 说明：不考虑需要联网的功能，聚焦离线单机体验优化

---

## 一、总体评价

**会下载，但不会长期玩。**

作为碎片化小游戏，打开即玩、无需联网的定位正确。但玩 10 分钟后很难有动力再打开第二次。

**核心问题：缺乏"再玩一次"的动力。**

---

## 二、当前最大问题（重新审视）

### 1. 收藏功能存在，但不够"香"

**现状：** 已有 CollectionPage（收藏页），用户可以收藏成语。

**问题：** 
- 收藏后只是列表展示，没有"收集成就感"
- 没有进度展示（"已收集 50/500"）
- 没有分类/整理功能
- 用户不知道为什么要收藏

**改进方向：** 把"收藏夹"升级为"成语图鉴"，增加收集成就感。

### 2. 闯关模式太残酷

连续答对 5 题通关，答错一题**全部重来**——这让人想摔手机。

应该至少有"生命值"机制（3条命），而不是一错全废。

### 3. 500 个成语重复太快

以每天玩 10 分钟计算，一周就把所有成语见完了，之后零重玩价值。

**但问题在于：** 不是成语数量少，而是：
- 没有"已玩过"标记，导致重复出题
- 没有"复习模式"，错过的成语没有针对性练习

### 4. 每日挑战题型单一

"基于日期种子随机"的设计是合理的（保证同一天所有人拿到同一套题），但只有"猜成语"一种题型，三天就腻了。

### 5. 没有成就/目标感

只有分数记录，没有"里程碑"——玩家不知道自己在为什么而努力。

## 四、详细改进方案

### 1. 成语图鉴系统（改造现有收藏功能）

**现状问题：** CollectionPage 只是简单的列表，没有收集动力。

**改进方案：**

#### 4.1.1 进度展示
```
我的收藏 (50/500)
████████░░░░░░░░░░░░ 10%
```

#### 4.1.2 分类整理
- 按难度分类（入门/进阶/大师）
- 按首字母分类（A/B/C...）
- 按标签分类（动物/数字/自然...）

#### 4.1.3 卡片样式优化
- 每个成语一张精美卡片
- 正面：成语 + 拼音
- 点击翻转：释义 + 出处 + 例句
- 卡片有"新收藏"标记（高亮显示）

#### 4.1.4 收集成就
- "初识成语"：收藏第一个成语
- "小有成就"：收藏 50 个成语
- "成语达人"：收藏 200 个成语
- "妙语大师"：收藏 500 个成语

---

### 2. 生命值系统（闯关模式）

**现状问题：** 连续答对 5 题通关，答错一题全部重来。

**改进方案：**

#### 4.2.1 3条命机制
- 初始 3 条命（❤️❤️❤️）
- 答错扣 1 条命
- 命为 0 时游戏结束
- 通关后奖励 1 条命

#### 4.2.2 续命道具（可选）
- 观看激励视频续命（如果需要广告变现）
- 或者用积分续命（如果有积分系统）

#### 4.2.3 难度递增
- 第 1-5 题：简单成语
- 第 6-10 题：中等成语
- 第 11-15 题：困难成语

---

### 3. 成就系统（离线版排行榜）

**现状问题：** 只有分数记录，没有里程碑。

**改进方案：**

#### 4.3.1 成就列表
| 成就名称 | 达成条件 | 奖励 |
|----------|----------|------|
| 🎯 初出茅庐 | 第一次答对成语 | 无 |
| 🔥 火力全开 | 连续答对 10 题 | 解锁"火焰"主题 |
| ⚡ 闪电手 | 10秒内答对5题 | 无 |
| 📚 博学多才 | 收藏 50 个成语 | 解锁"古籍"主题 |
| 🏆 不败战神 | 闯关模式通关 | 无 |
| 💎 成语大师 | 收藏 500 个成语 | 解锁"大师"主题 |

#### 4.3.2 成就展示页
- 网格布局展示所有成就
- 已解锁：彩色图标
- 未解锁：灰色图标 + 达成条件提示

---

### 4. 已玩成语标记（避免重复）

**现状问题：** 算法可能每次都从 500 个里随机，导致有些成语反复出现。

**改进方案：**

#### 4.4.1 本地记录已玩成语
```json
{
  "playedChengyu": ["龙飞凤舞", "一马当先", ...],
  "correctCount": 150,
  "wrongCount": 30
}
```

#### 4.4.2 智能出题算法
- 优先从未玩过的成语中随机
- 如果所有成语都玩过，进入"复习模式"
- 复习模式：优先出错的成语

#### 4.4.3 重置选项
- 设置页增加"清除游戏记录"按钮
- 清除后可重新玩所有成语

---

### 5. 复习模式

**现状问题：** 错过的成语没有针对性练习。

**改进方案：**

#### 4.5.1 复习模式入口
- 首页增加"复习模式"按钮
- 只出不题错过的成语

#### 4.5.2 复习统计
```
复习模式
已复习：30 题
正确率：80%
```

#### 4.5.3 掌握标记
- 连续答对 3 次，标记为"已掌握"
- 已掌握的成语不再出现在复习模式

---

### 6. 分享功能

**现状问题：** 成绩只能自己看，没有分享动力。

**改进方案：**

#### 4.6.1 分享成绩
- 游戏结束后，增加"分享成绩"按钮
- 生成成绩卡片（分数 + 用时 + 日期）
- 调用系统分享API（微信/朋友圈/QQ等）

#### 4.6.2 分享收藏
- 成语图鉴页，每个成语卡片可单独分享
- 生成"我今天学会了XX成语"卡片
- 包含成语释义、出处、例句

#### 4.6.3 分享卡片设计
- 古风设计，符合成语主题
- 包含App名称和二维码（吸引新用户）

---

## 五、最核心的建议

### 做"成语图鉴"系统 + 成就系统

现在用户答对成语后，只显示一个 ResultPage 就完了，成语信息转头就忘。

如果用户答对后可以**收藏这个成语到自己的图鉴**，图鉴里有：
- 进度展示（"已收集 50/500"）
- 分类整理（按难度/首字母/标签）
- 精美卡片展示（拼音、释义、出处、例句）
- 收集成就（"收藏 50 个成语"解锁成就）

这就能形成：

**收集 → 查看成就 → 分享 → 好友攀比 → 回流** 的闭环。

这是我作为一个用户**最想要的功能**。

## 六、开发优先级建议

### 第一阶段（快速见效）
1. **生命值系统** - 改动小，体验提升明显
2. **已玩成语标记** - 避免重复，延长寿命
3. **分享功能** - 增加传播性

### 第二阶段（核心功能）
1. **成语图鉴系统** - 改造现有收藏功能
2. **成就系统** - 增加目标感

### 第三阶段（增值功能）
1. **复习模式** - 增加重玩价值
2. **题型微调** - 同一题型不同难度
3. **成语典故小故事** - 增加趣味性
